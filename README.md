# zepto-sql-project
SQL project using PostgreSQL for inventory analysis
# SQL Inventory Management Project

## Objective
Designed a relational database to manage products, pricing, discounts, and stock availability.

## Tools Used
- PostgreSQL
- pgAdmin
- SQL

## Features
- Table creation with constraints
- Discounted price calculations
- Stock availability tracking
- Aggregate and join queries

## Files Included
- zepto_schema.sql
- sample_queries.sql
